A-START-TO-AWS_EKS

A guide Project to learn and explore Kubernetes on AWS • March 2020

This is a guide which I made while learning kubernetes for beginners to get started with Kubernetes on AWS, provisioning EKS cluster and working with the cluster to deploy different microservices. It emphasises on understanding all of the configurations (network, security, storage and so) and launching them using Iaas (cloudformations, kubernetes yaml config templates). It consists of some of the Administration works like Autoscaling ,Monitoring and so on.
